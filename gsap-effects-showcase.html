Typewriter Effect
TEXT

Hiệu ứng gõ chữ từng ký tự. Dùng GSAP TextPlugin để animate từng character một cách mượt mà. Phù hợp cho hero text, tagline.

<!-- HTML -->
<div class="typewriter-text" id="tw">|</div>
/* CSS */
.typewriter-text {
  font-size: 2rem;
  font-weight: 700;
  color: #7c6df5;
}
// JS — cần gsap + TextPlugin
gsap.registerPlugin(TextPlugin);
gsap.to("#tw", {
  duration: 2,
  text: {
    value: "Hello World!",
    delimiter: ""
  },
  ease: "none",
  repeat: -1,
  repeatDelay: 1,
  yoyo: true
});

Glitch Text Effect
TEXT

Hiệu ứng nhiễu loạn digital glitch. Kết hợp GSAP timeline với CSS pseudo-elements để tạo hiệu ứng tín hiệu bị lỗi hấp dẫn.

<div class="glitch"
     data-text="GLITCH">
  GLITCH
</div>
.glitch { position: relative; }
.glitch::before,
.glitch::after {
  content: attr(data-text);
  position: absolute;
  top: 0; left: 0;
  opacity: 0;
}
.glitch::before { color: #ff0077; }
.glitch::after  { color: #00ffff; }
const tl = gsap.timeline({repeat:-1, repeatDelay:2});
tl.to(".glitch::before", {
  opacity:1, x:-4, duration:0.1
})
.to(".glitch::after", {
  opacity:1, x:4, duration:0.1
}, "<")
.to(".glitch::before,.glitch::after", {
  opacity:0, x:0, duration:0.1,
  stagger:0.05
});

Number Counter
TEXT

Đếm số từ 0 đến mục tiêu với easing mượt mà. Dùng GSAP object tween để animate giá trị số, thường dùng cho statistics và dashboard.

<div id="counter">0</div>
#counter {
  font-size: 4rem;
  font-weight: 700;
  color: #f5a623;
  font-variant-numeric: tabular-nums;
}
const obj = { val: 0 };
gsap.to(obj, {
  val: 9999,
  duration: 2.5,
  ease: "power2.out",
  onUpdate: () => {
    document.getElementById("counter")
      .textContent = Math.round(obj.val).toLocaleString();
  }
});

Ripple Button
BUTTON

Hiệu ứng sóng lan ra từ điểm click. Tính toán vị trí click rồi dùng GSAP để animate div hình tròn từ 0 đến scale lớn với fade out.

<button class="ripple-btn"
  onclick="ripple(event)">
  CLICK ME
</button>
.ripple-btn {
  position: relative;
  overflow: hidden;
  padding: 12px 32px;
  border: 2px solid #7c6df5;
  background: transparent;
  cursor: pointer;
}
.ripple {
  position: absolute;
  border-radius: 50%;
  background: rgba(124,109,245,0.4);
  pointer-events: none;
}
function ripple(e) {
  const btn = e.currentTarget;
  const r = document.createElement("div");
  r.className = "ripple";
  const rect = btn.getBoundingClientRect();
  const size = Math.max(rect.width, rect.height) * 2;
  gsap.set(r, {
    width: size, height: size,
    x: e.clientX - rect.left - size/2,
    y: e.clientY - rect.top - size/2,
    scale: 0
  });
  btn.appendChild(r);
  gsap.to(r, {
    scale: 1, opacity: 0,
    duration: 0.6,
    ease: "power2.out",
    onComplete: () => r.remove()
  });
}

Magnetic Button
BUTTON

Button bị hút về phía con trỏ chuột khi di chuyển gần. Dùng GSAP quick setter để update transform cực nhanh mà không lag.

<button class="magnetic">
  Hover Me
</button>
.magnetic {
  display: inline-block;
  will-change: transform;
  cursor: pointer;
}
const btn = document.querySelector(".magnetic");
const setX = gsap.quickSetter(btn, "x", "px");
const setY = gsap.quickSetter(btn, "y", "px");

btn.addEventListener("mousemove", e => {
  const r = btn.getBoundingClientRect();
  const cx = r.left + r.width / 2;
  const cy = r.top + r.height / 2;
  setX((e.clientX - cx) * 0.4);
  setY((e.clientY - cy) * 0.4);
});

btn.addEventListener("mouseleave", () => {
  gsap.to(btn, { x: 0, y: 0, duration: 0.4 });
});

Shake Error Effect
BUTTON

Input rung lắc khi có lỗi validation. Dùng GSAP keyframes x animation để tạo cảm giác phản hồi tự nhiên, thay thế alert box.

<input id="field" type="text">
<button onclick="validate()">Submit</button>
#field {
  border: 2px solid #252535;
  border-radius: 8px;
  padding: 10px 16px;
}
#field.error {
  border-color: #f55252;
}
function validate() {
  const el = document.getElementById("field");
  el.classList.add("error");
  gsap.to(el, {
    keyframes: {
      x: [-10, 10, -8, 8, -4, 0]
    },
    duration: 0.5,
    ease: "power2.out"
  });
  setTimeout(() => {
    el.classList.remove("error");
  }, 1500);
}

Page Transition
TRANSITION

Overlay trượt vào rồi trượt ra để chuyển trang mượt mà. Kỹ thuật cổ điển trong web design dùng GSAP timeline với scaleY.

<div id="overlay"></div>
<a onclick="navigate('/page')">
  Go
</a>
#overlay {
  position: fixed;
  inset: 0;
  background: #7c6df5;
  transform: scaleY(0);
  transform-origin: bottom;
  z-index: 9999;
}
function navigate(url) {
  const tl = gsap.timeline();
  tl.to("#overlay", {
    scaleY: 1,
    duration: 0.6,
    ease: "power4.inOut",
    transformOrigin: "bottom"
  })
  .call(() => window.location.href = url)
  .to("#overlay", {
    scaleY: 0,
    transformOrigin: "top",
    duration: 0.6,
    ease: "power4.inOut"
  });
}
